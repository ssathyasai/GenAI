# Generative AI

This repository explores modern generative AI techniques, focusing on Large Language Models (LLMs) and Retrieval-Augmented Generation (RAG) systems.

## Contents

### Large Language Models
- **LLMs/** - Language model implementations including:
  - **Mini_LLM.ipynb** - Custom mini language model from scratch
  - **Fine-Tuning in LLM/** - Fine-tuning techniques for large language models

### Retrieval-Augmented Generation (RAG)
- **RAG/** - Complete RAG pipeline implementations:
  - **RAG_Pipeline.ipynb** - End-to-end RAG system
  - **RAG_ChromaDB.ipynb** - RAG implementation using ChromaDB vector database
  - **RAG Langchain/** - RAG systems built with LangChain framework
  - **Tasks/** - Practical RAG assignments

## Topics Covered
- Building language models from scratch
- Fine-tuning LLMs for specific tasks
- Vector databases and embeddings
- RAG architecture and implementation
- LangChain framework for LLM applications
- Document retrieval and generation systems
